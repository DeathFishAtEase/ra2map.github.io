【工具】CSF与JSON互相转换工具
 需要你会使用Python，文件在https://gist.github.com/lewisxy/1f96e60119f29b3b98cf6d7f234a870e
（国内镜像：https://ghproxy.com/https://gist.githubusercontent.com/lewisxy/1f96e60119f29b3b98cf6d7f234a870e/raw/ba6046f6aa031a1bbab0a56c8e8bc7625ab5604f/csftool.py）
用法：
-h, --help show this help message and exit
-d, --decode decode csf file (into JSON)
-e, --encode encode csf file (from JSON)
-i INPUT, --input INPUT
-o OUTPUT, --output OUTPUT
如 python csftool.py -d -i stringtable99.csf -o s.json

随机地图生成器
本生成器的原理就是拼图，在一个方形的地形区块中，定义每个边的连接方式，相同的连接方式相邻的边之间，其地形也是相连的，由此拼接成一张大地图
